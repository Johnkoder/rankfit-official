# RankFit

<img src="assets/mascot/fit-chan.png" alt="Fit-chan holding a dumbbell" width="160" align="right">

## Install on Android

1. Download the APK from the latest release.
2. Open the APK on your Android device.
3. If Android asks, allow installs from your browser or file manager.
4. Follow the installer prompts.

This first beta APK is debug-signed for testing. Android may show extra warnings because it is installed outside the Play Store.
